# Project N.O.M.A.D. Brasil

> Servidor offline de conhecimento, mapas, educação e IA local adaptado para usuários brasileiros.

O **Project N.O.M.A.D. Brasil** é uma adaptação brasileira do projeto open source **Project N.O.M.A.D.**, com foco em disponibilizar uma experiência mais acessível para usuários do Brasil, incluindo documentação em português, conteúdos offline em português brasileiro, mapas do Brasil e uso de IA local em português.

Este projeto nasce com o objetivo de tornar o conceito do N.O.M.A.D. mais útil para o público brasileiro, especialmente para cenários de estudo, infraestrutura, preparação emergencial, educação, uso offline, laboratórios domésticos e ambientes sem internet confiável.

---

## Objetivo do projeto

O objetivo do **Project N.O.M.A.D. Brasil** é manter uma versão paralela e comunitária voltada ao Brasil, sem depender diretamente da inclusão das alterações no projeto original.

A proposta inicial é:

- Manter compatibilidade com o projeto original;
- Facilitar a instalação para usuários brasileiros;
- Documentar tudo em português brasileiro;
- Incentivar o uso de conteúdos em português;
- Adicionar suporte e documentação para mapas offline do Brasil;
- Orientar o uso de IA local respondendo em português;
- Criar uma base para futura tradução da interface principal.
