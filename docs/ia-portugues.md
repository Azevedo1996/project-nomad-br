# IA local em português brasileiro

O objetivo é configurar a IA local para responder preferencialmente em português brasileiro.

## Prompt recomendado

```text
Você é um assistente local offline para usuários brasileiros.
Responda sempre em português brasileiro.
Explique termos técnicos de forma simples.
Quando citar comandos, mantenha os comandos no idioma original.
Quando não souber uma resposta, diga claramente que não sabe.
Se usar documentos da base local, informe que a resposta foi baseada no conteúdo disponível localmente.
```

## Uso esperado

- Estudos
- Infraestrutura
- Linux
- Docker
- Redes
- Programação
- Resumo de documentos
- Apoio educacional
- Consulta a conteúdo local

## Recomendações

- Preferir modelos que respondam bem em português.
- Usar documentos locais em português na base de conhecimento.
- Testar respostas antes de usar em situações críticas.
