# Release v0.1.0-br

> Primeira versão pública da adaptação brasileira do Project N.O.M.A.D.

A versão `v0.1.0-br` marca o início oficial do **Project N.O.M.A.D. Brasil** como um projeto paralelo, comunitário e voltado ao público brasileiro.

## Objetivo desta versão

Esta versão não altera profundamente a aplicação principal ainda. O foco é criar a base brasileira do projeto:

- documentação em português;
- Docker Compose brasileiro;
- estrutura de catálogos brasileiros;
- validação dos catálogos;
- automações iniciais;
- estrutura de mapas PMTiles do Brasil;
- preparação para conteúdos em português.

## Destaques

### Docker Compose brasileiro

Arquivo principal:

```text
docker-compose.br.yml
```

Portas padrão:

```text
8082 - Painel principal
9998 - Dozzle/logs
```

### Catálogos brasileiros

Pasta:

```text
collections/br/
```

Inclui:

```text
wikipedia-br.json
kiwix-categories-br.json
maps-br.json
catalog-index.json
```

### Documentação brasileira

Pasta:

```text
docs/
```

Principais guias:

```text
primeiros-passos.md
guia-usuario-brasil.md
faq-brasil.md
troubleshooting-brasil.md
contribuindo-brasil.md
```

### Workflows GitHub Actions

Workflows adicionados/adaptados:

```text
Validate Brazilian Catalogs
Validate Brazilian Maps Catalog
Update Brazilian Kiwix Metadata
Validate Collection URLs
```

### Mapas PMTiles do Brasil

Estrutura inicial:

```text
maps/br/
scripts/generate-br-pmtiles.sh
scripts/generate-br-pmtiles.ps1
scripts/update-br-maps-catalog.py
```

```bash
cd project-nomad-br
docker compose -f docker-compose.br.yml up -d
```

Acesse:

```text
http://localhost:8082
```
